# Лабораторная работа №1: Обобщённые квадратурные формулы Ньютона-Котесса

**Тема:** Численное интегрирование методами прямоугольников, трапеций, Симпсона и трёх восьмых

## 📌 Описание проекта

Данный репозиторий содержит реализацию четырёх классических методов численного интегрирования (квадратурных формул Ньютона-Котесса) на языке Python. Программа позволяет исследовать сходимость методов, анализировать погрешности и сравнивать эффективность различных подходов в зависимости от гладкости подынтегральной функции.

**Цели работы:**
1. Исследовать применимость методов и их свойства на тестовых задачах.
2. Решить конкретные задачи численного интегрирования и подтвердить достоверность решений численным экспериментом.
3. Проверить решения средствами Python и визуализировать результаты.

## 🧮 Реализованные методы

| Метод | Порядок точности | Функция | Требования к гладкости |
|-------|------------------|---------|-----------------|
| Прямоугольников (средних) | \(O(h^2)\) | `rectangles()` | C^1 |
| Трапеций | \(O(h^2)\) | `trapezoids()` | C^2 |
| Симпсона (парабол) | \(O(h^4)\) | `simpson()` | C^4, n чётное |
| Трёх восьмых | \(O(h^4)\) | `three_eights()` | C^4, n кратно 3 |
